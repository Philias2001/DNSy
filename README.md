## Project assignment
Make a program that will search all the dns records of a domain name.

## Build Go project
To build this Go project, clone the repo and type in the following command:
#### Windows
``` go
go build -o DNSy.exe
```
#### Mac/Linux
``` go
go build -o DNSy
```
